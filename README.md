# Customer Data Analysis & RFM Segmentation
<br>


# 📌 Project Overview

This project focuses on Customer Data Analysis using Python to generate valuable business insights through RFM (Recency, Frequency, Monetary) Analysis.
The goal of this project is to understand customer purchasing behavior, segment customers based on their transaction patterns, and help businesses improve customer retention and revenue strategies.

# 🎯 Objectives
Clean and preprocess customer datasets

Merge customer master and transaction datasets

Perform Exploratory Data Analysis (EDA)

Calculate RFM metrics:

1.Recency → How recently a customer purchased

2.Frequency → How often a customer purchases

3.Monetary → How much money a customer spends

4.Segment customers into business categories such as:

Champion

📌 Conclusion

This project demonstrates how customer transaction data can be transformed into actionable business insights using Python and RFM Analysis.
It helps businesses identify high-value customers, improve customer retention, and make data-driven decisions.

# 🔍 Project Workflow

1️⃣ Data Cleaning

Removed missing values

Checked duplicate customer IDs

Converted date columns into datetime format

Validated customer IDs across datasets

2️⃣ Data Merging

Merged customer and transaction datasets using:

pd.merge()

3️⃣ RFM Analysis

Calculated:

Recency

Frequency

Monetary Value

Generated RFM scores using:

pd.qcut()

4️⃣ Customer Segmentation

Customers were categorized into different business segments based on RFM scores.

5️⃣ Data Visualization

Created visualizations such as:

Customer Segment Distribution

Revenue by Segment

Recency vs Monetary Scatter Plot

Pareto Analysis (80/20 Rule)

# 📊 Key Insights

1.Loyal and Champion customers contribute the highest revenue.

2.Some customers require attention due to low recent activity.

3.A small percentage of customers generate the majority of revenue (Pareto Principle).

4.Customer segmentation helps businesses create targeted marketing strategies.
